# Stock Dashboard Project

This project analyzes historical stock and revenue data for **Tesla** and **GameStop** using Python, `yfinance`, `BeautifulSoup`, and `Plotly`.  
It was developed as part of the **Python Project for Data Science** final project.

## Key Features
- Extracts stock price data using the `yfinance` library  
- Scrapes quarterly revenue data with `BeautifulSoup`  
- Cleans and visualizes data using `pandas` and `plotly`  
- Builds an interactive dashboard for comparison
